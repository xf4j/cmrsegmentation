Supported formats: Cine (multi-phase 2D) or Scar/LGE (single-phase 2D). One DICOM file per slice/phase image. Multi-frame (composite/enhanced) DICOMs are not supported — use single-image DICOMs and load all of them at once. Parsing is based on DICOM header tags; file names do not matter. For additional format support, contact Dr. Xue Feng (xf4j@virginia.edu).

Output: Click "Download Results" to get a ZIP with NIfTI volumes (.nii.gz) and CSV measurements.

Try it: Click "Cine Sample" or "Scar Sample" above to download sample DICOM files. Unzip, then select all .dcm files in the DICOM file picker.

Reference: Feng X, Meyer CH. CPU-based fast cardiac cine MRI segmentation using lightweight CNNs and knowledge distillation. Journal of Cardiovascular Magnetic Resonance (under review).
